# Hospital Management System

A Spring Boot REST API application for managing 
hospital operations including patient records, 
doctor appointments, billing, and inventory management.

## Project Overview
This system streamlines day-to-day hospital operations 
by integrating patient management, doctor scheduling, 
billing, and inventory into one unified platform — 
improving efficiency and healthcare service delivery.

## Key Features
- Patient Management: Patient records, medical 
  history, and appointments
- Doctor Management: Doctor profiles and scheduling
- Billing System: Automated patient billing process
- Inventory Management: Medicine and equipment tracking

## Tech Stack
- Java 17
- Spring Boot — Backend framework
- Hibernate/JPA — ORM for database interaction
- MySQL — Primary database
- REST APIs — Backend communication
- Maven — Build tool

## How to Run

git clone https://github.com/gousejaheer-syed/hospital-management-api
cd hospital-management-api
mvn spring-boot:run

Access at: http://localhost:8080

## API Endpoints
- GET /patients — Get all patients
- POST /patients — Register new patient
- GET /doctors — Get all doctors
- POST /appointments — Book appointment
- GET /billing — Get billing records
